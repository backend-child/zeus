Getting Started with Zeus CLI

Your step-by-step guide to unleashing the power of Zeus in your DevOps workflow

Install Zeus CLI

pip install -g zeus-cli

Install Zeus CLI globally with a single command. This allows access to all cloud-debugging features from any terminal.

Connect to a Cloud Provider

zeus connect aws --access-key YOUR_KEY --secret YOUR_SECRET

Authenticate into AWS, GCP, Azure or DigitalOcean. Zeus stores and validates your credentials instantly.

Check Connection Status

zeus status

Quickly verify that your connection is active, secure, and ready to run queries or fetch assets.

Fetch Cloud Buckets or Tables

zeus fetch s3 --buckets

Retrieve your available cloud buckets or database tables in seconds. Works with S3, Firestore, Blob, and more.
